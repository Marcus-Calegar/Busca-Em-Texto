# Busca Em Texto

Aplicação Windows Forms em C# (.NET Framework 4.7.2) para demonstração de algoritmos de busca de padrão em texto. Projeto acadêmico para a disciplina de Algoritmos e Estruturas de Dados II.

Resumo
-------
O programa permite abrir arquivos de texto (.txt, .rtf), pesquisar por um padrão usando diferentes algoritmos (Força Bruta, KMP, Boyer-Moore, Rabin-Karp), navegar entre ocorrências, realçar resultados e substituir o texto encontrado.

Principais funcionalidades
-------------------------
- Abrir arquivos .txt e .rtf
- Pesquisar padrão com opção case-sensitive
- Substituir ocorrências encontradas
- Realçar todas as ocorrências e destacar a ocorrência atual
- Navegar entre ocorrências (anterior/próximo)

Algoritmos implementados
------------------------
- Força Bruta
- KMP (Knuth-Morris-Pratt)
- Boyer-Moore
- Rabin-Karp

Como usar
---------
1. Execute o programa (`BuscaTexto.exe`).
2. Menu -> Abrir para carregar um arquivo de texto (.txt ou .rtf).
3. Use o menu de pesquisa para escolher o algoritmo desejado.
4. No diálogo de pesquisa informe o padrão, marque "case sensitive" se desejar distinção entre maiúsculas/minúsculas. Ative "substituir" para informar o texto substituto.
5. Use os botões "Proximo" e "Anterior" para navegar entre as ocorrências encontradas.
